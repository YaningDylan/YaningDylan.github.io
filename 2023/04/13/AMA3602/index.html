Chapter 1: Simple linear Regression

1. Model

\[ y = \beta_0 + \beta_1 x + \varepsilon \]

1.1 Assumption for SLR

  1. Linearity

  2. Unbiased system

    \(E(\varepsilon) = 0\)

    \(\varepsilon \sim N(0,\sigma^2)\)

  3. Errors are independent

  4. Errors follows normal distribution

    \(\varepsilon \sim N(0,\sigma^2)\)

1.2 Targets to realize SLR

  1. estimate \(\beta_0\) and \(\beta_1\)

  2. estimate \(\sigma^2\)

  3. hypothesis testing on \(\beta_0\) and \(\beta_1\)

  4. Confidence Interval of \(\beta_0\) , \(\beta_1\) and mean response \(E(Y|x)\)

  5. Prediction interval of new observations

Target 1 : Estimation of \(\beta_0\) and \(\beta_1\)

  • method : Least-squares estimation (LSE)

  • estimator : \(\hat \beta_0\) and \(\hat \beta_1\)

  • process

    Therefore, the least-squares normal equations are \[ \begin{align} n \hat \beta_0 + \hat \beta_1 \sum^n_{i=1}x_i &= \sum^n_{i=1}y_i\\ \hat \beta_0 \sum^n_{i=1} x_i + \hat\beta_1 \sum^n_{i=1} x^2_i &= \sum_{i=1}^n y_i x_i \end{align} \]

  • solution \[ \hat{\beta}_1=\frac{S_{x y}}{S_{x x}}=\left(\frac{S_{x y} / n}{S_{x x} / n}\right)\\[6pt] \hat{\beta}_0=\bar y - \hat\beta_1\bar x \\[6pt] \]

  • Additional formula \[ \bar{y}=\frac{1}{n} \sum_{i=1}^n y_i,\quad \bar{x}=\frac{1}{n} \sum_{i=1}^n x_i,\\[6pt] S_{x y}=\sum_{i=1}^n\left(y_i-\bar{y}\right)\left(x_i-\bar{x}\right)=\sum_{i=1}^n y_i\left(x_i-\bar{x}\right)\\[6pt] S_{x x}=\sum_{i=1}^n\left(x_i-\bar{x}\right)^2 \]

    • Proof of \(S_{xy}\) and \(S_{xx}\)

Target 2 : Estimation of \(\sigma^2\)

Since \(\sigma^2\) is computed from the regression model residuals, we say that it is a model-dependent estimate of \(\sigma^2\)

  • Condition

  • estimator: \(\sigma^2\)

  • process:

  • solution \[ \sigma^2 = \frac{SS_{Res}}{n-2} = MS_{Res} \]

  • Additional formula

    • Residual sum of squares : \(SS_{Res}\)

    • Residual mean square : \(MS_{Res}\)

    • standard error of regression : \(\sigma^2\) \[ \sigma^2 = MS_{Res} = \frac{SS_{Res}}{n-2} \\[6pt] SS_{Res}= \sum^n_{i=1}e_i^2 =\sum_{i=1}^n(y_i-\hat y_i)^2 \]

Target 4 : Confidence Interval on \(\beta_0\), \(\beta_1\), \(\sigma^2\) and Mean Response
  1. Confidence Interval on \(\beta_0\) and \(\beta_1\)

    • condition

    • request : Confidence Interval on \(\beta_0\) and \(\beta_1\)

    • process:

      \[ \hat \beta_1 \sim N\left(\beta_1,\frac{\hat \sigma^2}{S_{xx}}\right)\\ \hat \beta_0 \sim N \left(\beta_0\ ,\ \hat \sigma^2\left( \frac{1}{n}+\frac{\bar x^2}{S_{xx}} \right) \right) \]

    • solution \[ \text{Confidence interval on $\beta_1$ is}\\ \hat{\beta}_1-t_{\alpha / 2, n-2} \operatorname{se}\left(\hat{\beta}_1\right) \leq \beta_1 \leq \hat{\beta}_1+t_{\alpha / 2, n-2} \operatorname{se}\left(\hat{\beta}_1\right)\\ \text{Confidence interval on $\beta_0$ is}\\ \hat{\beta}_0-t_{\alpha / 2, n-2} \operatorname{se}\left(\hat{\beta}_0\right) \leq \beta_0 \leq \hat{\beta}_0+t_{\alpha / 2, n-2} \operatorname{se}\left(\hat{\beta}_0\right)\\ \]

    • additional fomula \[ \operatorname{se}\left(\hat{\beta}_1\right) = \sqrt{\frac{\hat \sigma^2}{S_{xx}}} = \sqrt{\frac{MS_{Res}}{S_{xx}}} \\ \operatorname{se}\left(\hat{\beta}_0\right) = \sqrt{\hat \sigma^2\left( \frac{1}{n}+\frac{\bar x^2}{S_{xx}} \right)} \]

  2. Confidence Interval of \(\sigma^2\)

    • condition

    • request : Confidence Interval of \(\sigma^2\)

    • process :

      \[ \frac{SS_{Res}}{\sigma^2} \sim \chi^2_{n-2} \]

    • solution \[ \text{Confidence Interval on }\sigma^2 \\ \frac{(n-2)MS_{Res}}{\chi^2_{\alpha/2,n-2}}\leq \sigma^2 \leq \frac{(n-2)MS_{Res}}{\chi^2_{1-\alpha/2,n-2}} \]

  3. Interval Estimation of the Mean Response

    • condition

    • request : CI of $ $

    • process :

      Because mean response follows: \[ E(y|x_0)=\mu_{y|x_0} \sim N\left(\hat \beta_0+\hat \beta_1x_0 \ ,\ \sigma^2 \left[\frac{1}{n} + \frac{(x_0-\bar x)^2}{S_{xx}} \right] \right)\\ \] Therefore, \[ \frac{\hat \mu_{y|x_0} - \mathrm{E}\left(y | x_0\right)} {\sqrt{MS_{Res} \left( \frac{1}{n} + \frac{(x_0-\bar x)^2}{S_{xx}} \right) }} \sim t_{n-2}\\ \] Where, \[ \hat \mu_{y|x_0} = \hat \beta_0 + \hat \beta_1x_i \]

    • solution \[ \hat{\mu}_{y \mid x_0}-t_{\alpha / 2, n-2} \sqrt{M S_{R e s}\left(\frac{1}{n}+\frac{\left(x_0-\bar{x}\right)^2}{S_{x x}}\right)} \leq \mathrm{E}\left(y \mid x_0\right) \leq \hat{\mu}_{y \mid x_0}+t_{\alpha / 2, n-2} \sqrt{M S_{R e s}\left(\frac{1}{n}+\frac{\left(x_0-\bar{x}\right)^2}{S_{x x}}\right)} \]

  3. Gauss-Markov theorem

    For the regression model with the assumptions \(E(\varepsilon_i) = 0\), \(Var(\varepsilon_i) = \sigma^2\) and uncorrelated errors, the least-squares estimators are unbiased and have minimum variance when compared with all other unbiased estimators that are linear combinations of the \(y_i\).

    \[ Var(\hat\beta_{i \ LSE}) \leq Var(\hat\beta_i)\\[5pt] E(\hat{\beta}_1)=\beta_1 \\ Var(\hat{\beta}_1)=\frac{\sigma^2}{S_{xx}} \]

  4. BLUE

    best linear unbiased estimators, where“best” implies minimum variance when compared with all other unbiased estimators that are linear combinations of the \(y_i\)

2. Analysis of variance (ANOVA) (For F test)

  • Fundamental ANOVA identity \[ \sum^n_{i=1}(y_i-\bar y_i)^2 = \sum^n_{i=1}(\hat y_i-\bar y_i)^2 + \sum^n_{i=1}(y_i-\hat y_i)^2 \\[6pt] SS_T = SS_R + SS_{Res} \]

  • Additional formula

    • Total sum of squares : \(SS_T\)

      variation around mean

    • Regression/Model sum of squares : \(SS_R\)

      variation between regression line and mean

    • Residual/Error sum of squares : \(SS_{Res}\)

      variation around regression line

    \[ SS_T =\sum^n_{i=1}(y_i-\bar y_i)^2 \\ SS_R=\sum_{i=1}^n(\hat{y}_i-\bar{y})^2=\hat\beta_1S_{xy}\\ SS_{Res}=\sum_{i=1}^n (y_i-\hat{y}_i)^2\\ \]

  • degree of freedom \[ df_T=df_R+df_{Res}\\ df_T = n-1\\ df_R = 1\\ df_{Res} = n-2 \]
